Usage: nines analyze [OPTIONS]

  Analyze and decompose collected knowledge into structured units.

Options:
  --target-path PATH              Path to a repository or directory to analyze
                                  for Agent impact.  [required]
  --strategy [functional|concern|layer|graph]
                                  Decomposition strategy.  [default:
                                  functional]
  --output-dir PATH               Directory to write analysis results to.
  --agent-impact / --no-agent-impact
                                  Run Agent impact analysis (default:
                                  enabled).  [default: agent-impact]
  --keypoints / --no-keypoints    Extract key points (default: enabled,
                                  implies --agent-impact).  [default:
                                  keypoints]
  --depth [shallow|deep]          Analysis depth.  [default: shallow]
  --strict-graph / --no-strict-graph
                                  Exit non-zero when the knowledge-graph
                                  verification fails with any critical issue.
                                  Default: True for --strategy graph, False
                                  for the other strategies (which do not
                                  produce a graph).
  --audit / --no-audit            Run the C10 cross-artifact consistency
                                  auditor over the report (advisory by
                                  default).  Use --strict-audit to make
                                  critical findings block CI.  [default:
                                  audit]
  --strict-audit                  Exit non-zero (code 3) when --audit produces
                                  any critical finding.  Off by default —
                                  auditor is advisory-only.
  --help                          Show this message and exit.
