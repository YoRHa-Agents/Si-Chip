Usage: nines self-eval [OPTIONS]

  Run self-evaluation across all capability dimensions.

Options:
  --baseline-version TEXT         Version tag for this evaluation (used for
                                  baseline comparison).
  --output-dir PATH               Directory to write the self-evaluation
                                  report to.
  --project-root PATH             Project root directory for coverage
                                  measurement.
  --src-dir PATH                  Source directory for module/docstring/lint
                                  analysis.
  --test-dir PATH                 Test directory for test discovery.
  --capability-only               Run only capability evaluators, skip hygiene
                                  (faster iteration).
  --samples-dir PATH              Sample eval directory for
                                  EvalCoverageEvaluator.
  --golden-dir PATH               Golden test set directory for V1 scoring
                                  evaluators (D01/D03/D05).
  --metrics-config FILE           Path to a TOML file overriding the bundled
                                  C08 metric weights/thresholds (default:
                                  src/nines/data/self_eval_metrics.toml).
  --evaluator-timeout FLOAT       Per-evaluator wall-clock budget in seconds
                                  (C04).  Evaluators that exceed this budget
                                  are recorded with status='timeout' in the
                                  report and the run continues.  [default:
                                  60.0]
  --breakdown / --no-breakdown    C12: emit AgentBoard-style sub-skill
                                  breakdown panels alongside the flat
                                  dimension report.  Default off for backward
                                  compat.  [default: no-breakdown]
  --breakdown-format [text|json|markdown]
                                  C12: render the breakdown as ``text``,
                                  ``json``, or ``markdown``.  [default: text]
  --help                          Show this message and exit.
