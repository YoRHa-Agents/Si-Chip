Usage: nines [OPTIONS] COMMAND [ARGS]...

  NineS — Multi-vertex evaluation, collection, analysis, and self-iteration
  system.

Options:
  --version                       Show the version and exit.
  -c, --config PATH               Path to nines.toml config file.
  -v, --verbose                   Enable verbose output.
  -q, --quiet                     Suppress non-error output.
  -o, --output PATH               Write primary output to file instead of
                                  stdout.
  -f, --format [text|json|markdown]
                                  Output format.
  --no-color                      Disable colored output.
  --help                          Show this message and exit.

Commands:
  analyze    Analyze and decompose collected knowledge into structured...
  benchmark  Full analysis→benchmark→evaluate→mapping workflow.
  collect    Search and collect information from configured sources.
  eval       Run evaluation benchmarks on agent capabilities.
  install    Install or uninstall NineS as an agent skill.
  iterate    Execute a self-improvement iteration cycle.
  self-eval  Run self-evaluation across all capability dimensions.
  update     Check for and install NineS updates, refresh skill files.
