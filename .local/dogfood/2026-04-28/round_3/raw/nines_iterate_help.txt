Usage: nines iterate [OPTIONS]

  Execute a self-improvement iteration cycle.

Options:
  --max-rounds INTEGER  Maximum number of iteration rounds.  [default: 5]
  --threshold FLOAT     Convergence variance threshold.  [default: 0.05]
  --project-root PATH   Project root directory.
  --src-dir PATH        Source directory for analysis. Auto-detected if not
                        set.
  --test-dir PATH       Test directory. Auto-detected if not set.
  --samples-dir PATH    Sample eval directory for EvalCoverageEvaluator.
  --golden-dir PATH     Golden test set directory for V1 scoring evaluators.
  --gates / --no-gates  Run C07 quality gates against the final iteration
                        report. Gates are advisory unless --strict-gates is
                        passed.  [default: gates]
  --strict-gates        Promote gate failures from advisory warnings to
                        blocking errors. Implies --gates; exits non-zero when
                        any gate blocks.
  --help                Show this message and exit.
